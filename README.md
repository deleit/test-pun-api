<h1>API em C#</h1>

WEB API construída com .NET Framework. Acessa um banco de dados SQL Server e retorna um trocadilho ruim (ou bom, talvez) com o seguinte formato:
```
[
  {
    "Id": 1, 
    "Question": "Qual a banda favorita do leão?", 
    "Answer": "Rouge"
  }
]
```

<s>Disponível em: https://punapi.azurewebsites.net/api/Pun

Aceita como o parâmetro o Id de algum trocadilho: https://punapi.azurewebsites.net/api/Pun/1

Página de apresentação simples: https://deleit.github.io/pun-api-frontpage/</s>
